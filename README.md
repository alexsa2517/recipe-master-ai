# 🍳 Recipe Master AI

AI-powered cooking recipe platform for Thai, Chinese, and Japanese cuisine  
Designed to scale from **500 → 10,000+ recipes**

---

## 🚀 Version
**v0.1 — MVP (Launch)**

---

## 🎯 Project Goal

Build a scalable recipe application that helps users:

- Find recipes by ingredients
- Search meals by category
- Learn cooking step-by-step
- Save favorite recipes
- Support future AI cooking assistant

---

## 🧩 MVP Features (v0.1)

- 📚 500 high-quality recipes
- 🔍 Basic search (name / ingredient)
- 🗂️ Category filtering
- ❤️ Favorite system
- 🌏 Multi-language structure ready (TH / EN / CN / JP)
- 📱 Simple mobile UI (Flutter planned)

---

## ❌ Not Included (Future Versions)

- AI chat assistant
- Video recipes
- Social system (reviews / followers)

## 🗄️ Core Database Concept

Designed to support **10,000+ recipes without redesign**

Main tables:

- recipes
- ingredients
- recipe_ingredients
- recipe_steps
- categories
- countries
- tags
- users
- favorites

---

## 🍱 Recipe Standard

Each recipe includes:

- Title (multi-language ready)
- Ingredients (gram/ml standard)
- Step-by-step cooking instructions
- Cooking time
- Difficulty level
- Calories & nutrition info
- Tags (spicy, healthy, quick, etc.)
